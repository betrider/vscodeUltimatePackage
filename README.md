    2. vsce package // 로컬 빌드
    3. vsce publish // 아예 마켓까지 올려버리는 명령어

# Installation

- Open VS Code and click on Extensions Icon in the Activity Bar.
- Type **Ultimate Flutter Extension Pack**
- Install the Extension Pack.
